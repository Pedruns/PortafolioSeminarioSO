Actividad 1: El hardware de la computadora y el sistema operativo

Descripcion

En esta actividad se analiza cómo una computadora procesa datos mediante dos flujos principales: datos e instrucciones. Se revisan los componentes esenciales de la arquitectura Von Neumann, como la unidad aritmética, la memoria principal, la unidad de control y la unidad de entrada/salida. Además, se identifican los elementos de hardware y software que intervienen durante el encendido de una computadora, desde el arranque del hardware hasta la carga del sistema operativo. Finalmente, se hace uso de herramientas como ChatGPT para complementar la comprensión de estos procesos.

Actividad 2: Sistemas Operativos y sus tipos

Descripcion

En esta actividad se revisa la evolución de los sistemas operativos y su papel como intermediarios entre el hardware y las aplicaciones. Se busca comprender cómo estos sistemas se adaptan a las necesidades tecnológicas actuales. Para lograrlo, se realiza un cuadro comparativo que analiza los diferentes tipos de sistemas operativos y sus principales características, permitiendo al estudiante identificar y comparar sus funciones y clasificaciones.

Actividad 3: Configuración de dispositivos

Descripcion

La configuración y administración de dispositivos es fundamental para gestionar el hardware y las entradas y salidas que utiliza un sistema operativo. Aunque los dispositivos Plug and Play se configuran de manera automática en la mayoría de los casos, existen situaciones en las que es necesario ajustar manualmente ciertos parámetros para cumplir con tareas específicas. Por ello, se analiza el proceso necesario para preparar un dispositivo que permita la instalación de un sistema operativo, recopilando los requisitos indispensables para generar un dispositivo de arranque.

Actividad 4: Dispositivos físicos

Descripcion

Conocer y entender los dispositivos físicos que conforman una computadora es esencial, ya que forman parte de la gestión de entradas y salidas que realiza el sistema operativo. Estos dispositivos se clasifican según su función, como periféricos, de almacenamiento y de comunicación, por lo que identificar sus características y tipos permite una mejor comprensión del funcionamiento general del equipo.

Actividad 5: Concepto de máquina virtual y partición de disco

Descripcion

Una máquina virtual (VM) es un entorno que emula un sistema informático, con su propia CPU, memoria, red y almacenamiento, pero se ejecuta sobre hardware físico. Al permitir la virtualización del hardware, una VM puede ejecutar múltiples sistemas operativos de manera simultánea, o incluso funcionar como un proceso dentro de un sistema operativo. Esta capacidad de abstraer el hardware ofrece ventajas significativas, como la ejecución de diferentes sistemas operativos en un mismo equipo. En esta actividad se utilizará un hipervisor o Virtual Machine Monitor (VMM), que permite gestionar y ejecutar varias máquinas virtuales sobre un solo sistema físico.

Actividad 6: Distribuciones y licencias en un sistema operativo

Descripcion

El software se clasifica según las funciones que realiza, dividiéndose en dos grandes categorías: el software de base o sistemas, y el software de aplicación. El primero, que incluye al sistema operativo, se encarga de gestionar los recursos de la máquina, controlar sus actividades y facilitar la interacción con el usuario. Por otro lado, el software de aplicación está diseñado para satisfacer necesidades específicas de los usuarios finales. Además, una distribución de software agrupa programas que han sido compilados y configurados, y puede adoptar licencias como GPL o Open Source. Se exploran las distintas distribuciones, su licenciamiento y clasificación para entender mejor su uso y aplicación.
